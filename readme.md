# Code class: Debugging javascript with Chrome Devtools

## Install

```sh

# clone repository
$ git clone git@bitbucket.org:voorhoede/code-class-debug.git

# go inside
$ cd code-class-debug

# install dependencies
$ npm install

# start development and dev server. http://localhost:3000
$ npm start
```

Debug!
